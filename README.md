* NOTE: database needs to be migrated with '$ bin/rails db:migrate' before web app can work

Default is on port 3000.

'$ bin/rails server'

Run the console for Rails.

'$ bin/rails console'

Check the routes the app responds to.

'$ bin/rails routes'
